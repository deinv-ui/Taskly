# Taskly

**Taskly** is a modern, intuitive task management application built with **Nuxt 3**. It helps you organize, prioritize, and track your tasks efficiently, making your daily workflow smoother and more productive.

## Features
- Create, edit, and delete tasks
- Set task priorities (High, Medium, Low)
- Track task status (To-Do, In Progress, Completed)
- Interactive dashboard with task columns
- Task details modal for quick edits

## Getting Started

### Prerequisites
- Node.js >= 18
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd taskly
   ```
2. Install dependencies:
```bash
npm install
# or
yarn
```
3. Run the development server:
```bash
npm run dev
# or
yarn dev
```
4. Open your browser at http://localhost:3000 to view the app.
